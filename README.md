#### 1.项目介绍

技术栈+环境：SpringBoot + jsp + mysql5.7及以上 + maven3 + idea2022

系统角色：管理员、普通用户（学生）

系统功能：管理员（座位预订管理、座位使用统计、自习室管理、违规与投诉、学校大楼管理、预约时间管理、公告管理等）、学生（座位预订信息、座位预订、咨询与投诉等）
